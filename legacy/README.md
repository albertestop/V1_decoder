# Legacy Prototypes

This folder contains the original exploratory scripts before project refactoring.

- `orig_vae.py`: initial image compression + reconstruction evaluation script.
- `develop_files/`: early VAE experiments on sample image and `.npy` frame data.
- `neural_compression_main.py`: empty placeholder from initial repo state.

These files are kept for provenance and comparison. New development should happen under `src/` and `scripts/`.
